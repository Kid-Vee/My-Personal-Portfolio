# Portfolio Website

Personal portfolio site for **Gideon Vision Olufeagba** — Product Manager (CPM, AWS CCP).

Built with [Next.js](https://nextjs.org) (App Router), React 19, TypeScript, and Tailwind CSS v4.

## Getting Started

Install dependencies and run the dev server:

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view it.

The single-page app lives in [`app/portfolio.tsx`](app/portfolio.tsx) and is rendered by [`app/page.tsx`](app/page.tsx). Edit and the page hot-reloads.

## Scripts

- `npm run dev` — start the dev server (Turbopack)
- `npm run build` — production build
- `npm run start` — serve the production build
- `npm run lint:fix` — run ESLint with autofix

## Deploy

Deploys cleanly to [Vercel](https://vercel.com/new). See the [Next.js deployment docs](https://nextjs.org/docs/app/building-your-application/deploying).
